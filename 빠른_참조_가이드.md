
3. **확인**
   - 2-3분 후 라이브 사이트 확인
   - 브라우저 새로고침 (Ctrl+F5)

## 🆘 문제 해결 (응급처치)

### ❌ 사이트 404 오류
```bash
quick-deploy.bat
# 2-3분 기다리기
```

### ❌ 이미지 안 보임
```bash
# 1. 파일 확인
dir "frontend\public\images\image.jpg"
# 2. 재배포
quick-deploy.bat
```

### ❌ 변경사항 반영 안됨
```bash
# 강제 재빌드
rmdir /s /q "frontend\build"
npm run build --prefix frontend
npm run deploy --prefix frontend
```

### ❌ 서버 시작 안됨
```bash
# 포트 사용 중인 프로세스 종료 후
start-frontend.bat
```
